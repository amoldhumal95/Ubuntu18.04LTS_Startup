OPEN TERMINAL -> Enter the following commande to get updated and some tools

                  sudo bash

                  sudo apt-get update --fix-missing && sudo apt-get upgrade --fix-missing

                  sudo apt-get install gnome-tweaks

                  sudo apt-get install plank
